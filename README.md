# Fake News Detection Project

This repository contains the code for our fake news detection project for CS 229 - Machine Learning.

## Data Preprocessing
The NewsDataset folder contains the script we used to process the raw data 'True.csv' and 'Fake.csv'. These raw datasets are from the WELFake dataset.

## Model Architectures
The architecture for the models we worked on are located in src/models

## Dataset Management
We store classes that contain the specific format of each model's dataset in src/data

## Training and Evaluation Scripts
The training and evaluation scripts for each model are located in src/training

## Results
All of our result plots are located in NewsResults
